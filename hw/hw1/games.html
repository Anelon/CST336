Games!

The faster web language.

A faster language on the web means greater potentail for those browser games. No I am not talking about those silly flash games, there is really potential in this for tripple A quality games.

  1. The Epic Zen Garden demo is one of the coolest showcases of what this can do but there are a few other games
  2. Another is a port of Arx Fatalis
  3. Lastly is one that I have made myself, Kawaii Squid

There is also articles about porting Doom 3 to web assembly showing that there is further interest for developers to port their games.

Bullet.js is a bullet simulator library written in c++ compiled to Web Assembly for quick use on the web.
